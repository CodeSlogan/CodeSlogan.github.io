SSH公钥

这里我解释一下这个公钥和Git之间的关系。每台电脑上都可以安装git，从而产生对应的公钥。

gitee/github接收从git提交的项目代码，那问题就来了，如何把git和gitee/github连接起来呢？

就是靠这个公钥。公钥先由本地git产生，然后再将公钥信息提交到码云SSH中，便可实现连接
